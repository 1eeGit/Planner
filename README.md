# Planner
java final project for Programming II course

Functions: 1. allow user to make daily plan 
           2. record daily activities 
           3. calendar 
           4. goal setting

Objectives: User could review and compare execution rates, using the calendar to track deadlines and important events. The whole design focuses on improving time management and event tracking. The app would have a simple, clean and user-friendly style and a bright UI interface, and data would be saved and updated to a local file.
